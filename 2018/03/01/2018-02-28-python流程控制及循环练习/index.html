流程控制

if 条件 ：
执行代码块
elif 条件：
执行代码块
else:
执行代码块

例子代码一：

#inp变量接收用户输入的内容，input与用户交互，python2.7系列中是: raw_input
inp = input (“>>>”)
if inp == “1”: #如果用户输入1 ，控制台输出 你好1
print(“你好1”)
elif inp == “2”: #如果用户输入2 ，控制台输出 你好2
print(“你好2”)
elif inp == “3”: #如果用户输入3 ，控制台输出 你好3
print(“你好3”)
else : #如果用户输入非1，2，3 ，控制台输出wrong
print(“wrong”)

#输出除7以外的1-10的数值

#输出1-10，当数值为7时，跳过这个循环，继续输出以下循环数值
实例代码如下：
i = 1
while True:
if i == 7:
i = i + 1
continue;
print(i)
if i == 10:
break;
i = i + 1

#输入1 -100和的数值

#赋值变量i = 1,sum =0,第一个循环sum =0 + 1 ,执行i = i + 1 ,这时候i =2,再执行sum =sum +i (sum =1 ,而i=2,).

#最终sum = 1+ 2 ,如此循环，当i =100时，执行 break,中断整个循环，并执行print(sum)
实例代码如下：
i = 1
sum = 0
while True:
sum = sum + i
if i == 100:
break;
i = i + 1
print(sum)

#输出100以内的奇数

#要输出100以内的奇数，就得先明白什么时奇数，在数学中，一个数除以2 ，余数为1的数，就为奇数。

#输出1-100的数值，当遇到是奇数时就输入，不是奇数就不输出
实例代码如下：
i = 1 #给i赋值为1
while i < 101:
temp = i % 2
if temp ==1:
print(i)
else:
pass
i +=1
print(“执行结束”)

#输出100以内的偶数
实例代码如下：
i = 1 #给i赋值为1
while i < 101:
temp = i % 2
if temp ==0:
print(i)
else:
pass
i +=1
print(“执行结束”)

#输出1-2+3-4….99的和

#先输出1-99的和，如果是偶数sum - i ,如果是奇数，sum +i ,最后循环结束，输出sum的数值

sum =0
while i < 100:

if temp ==0:
sum = sum - i
else:
temp == 1
